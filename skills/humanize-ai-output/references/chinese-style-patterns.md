# Chinese Style Patterns

Use this reference when Chinese output sounds stiff, slogan-like, translated, or too visibly generated.

## High-Signal Patterns

| Pattern | Symptoms | Why it feels AI-ish | Better move |
|---|---|---|---|
| 强对照句式滥用 | 高频出现“不是……而是……”“是……而不是……” | 连续纠偏会显得用力，读者抓不到真正重点 | 只在澄清概念边界时保留；其他地方直接陈述，或用“更准确地说”“重点在于”“相比之下” |
| 官样抽象词 | 赋能、抓手、闭环、生态、底层逻辑、价值沉淀、提质增效 | 词很大，但缺少谁做什么、改变什么 | 改成具体动作、对象、指标、负责人、时间点 |
| 机械连接 | 首先/其次/最后、综上所述、值得注意的是 | 短文被写成考试作文或咨询模板 | 按内容关系连接：问题卡在哪、证据是什么、下一步看什么 |
| 过度排比 | 从 A 到 B，从 C 到 D；既要 A，也要 B，更要 C | 节奏整齐但信息密度低 | 保留最关键的一组，其他改成普通句或删掉 |
| 翻译腔 | 进行一个分析、有一个影响、帮助我们去理解 | 中文动词被名词化，句子拖沓 | 用动词直达：分析、影响、有助于理解 |
| 空泛建议 | 加强沟通、优化流程、持续迭代、提升效率 | 可适用于任何公司和任何项目 | 写频率、负责人、输入、产出、验收方式 |
| 情绪标签 | 温暖而有力量、令人深受启发、充满希望 | 先贴情绪，缺少场景支撑 | 用细节承载情绪，少用总结性标签 |
| 安全套话 | 可能、或许、在一定程度上、需要综合考虑 | 不敢判断，像在规避责任 | 保留真实不确定性，并说明决定变量 |
| 标题套路 | 当 X 遇见 Y、重新定义 X、一场关于 Y 的 Z | 像自动生成的营销号标题 | 改成对象 + 变化/冲突/结论 |
| 过度精修 | 每句都太干净、太锋利，像“不能...更关键...重点看...”连续推进 | 虽然比套话强，但仍像模型在压缩观点 | 保留一点口语颗粒感，例如“呢”“总之”“一来”，或把判断合成一条更自然的长句 |
| 过度顺滑 | 词序总是标准、书面、无停顿，例如“本周主要推进了三件事” | 正确但像编辑器统一润色过 | 在符合用户语感时保留轻微口语词序，例如“本周主要我们推进有三件事”“现在卡在”“可能就要” |
| 否定开头 | 先说“不要/不需要/不建议”，再给真正建议 | 像在纠正读者，容易显得模板化 | 直接给肯定建议；需要否定时要说清楚具体否定对象 |
| 冒号解释 | 高频使用“标题：内容”“问题：解释”“指标：解释” | 看起来像生成式提纲 | 改成直接句，或用破折号连接真正的指标名 |

## Rewrite Moves

- 把“我们将持续优化服务流程”改成“今天 18:00 前给出处理结果；如果需要补发，我们会在明天中午前完成。”
- 把“该公司有望受益于行业趋势”改成“判断要落到两个变量：订单能否从样机转批量，以及毛利率是否守住 35%。”
- 把“自律不是压抑自己，而是成就更好的自己”改成“自律最难的地方，其实是别把每天都安排满。能坚持的计划通常没那么漂亮。”
- 如果用户偏好更口语的中文，可以保留轻微停顿和连接：把“现在只留一个动作”改成“现在呢我只保留一个动作”；把“自律这件事，先别追求热血”改成“总之自律呢，一来不要只追求热血”。
- 项目周报可以保留自然的时间副词和个人动作：把“我会今天把待确认项列给财务和风控，明天下午同步结果”改成“我今天会把待确认项列给财务和风控，明天下午一起同步结果”。
- 投研或商业分析不要把所有句子都改成短促判断。可以把“不能只看...更关键的是...重点看...”合成更顺的一段：“行业 beta现在可能不合适了，要看看这些订单能否...那么估值上就很难...风险也还要看...”。
- PPT 或建议页避免用“先别...”做标题，直接写肯定动作，例如“寻找进度卡壳点”。
- 指标句少用“xxxx：xxxx”，可以写成“30天重点关注指标—交接消耗时间”。
- 把“项目稳步推进”改成“登录模块已联调完成，支付回调还差风控确认，预计周三给测试包。”

## Natural Connectors

Use lightly. Do not stack them.

- 更准确地说
- 重点在于
- 放到实际场景里
- 问题卡在
- 接下来要看
- 比较现实的做法是
- 这个判断有个前提
- 如果只看现在的信息
- 现在呢
- 总之
- 一来
- 以免
- 现在
- 可能就要
- 那么
- 还要
- 一起

## Final Check

- 是否保留了事实、数字、人物、时间和限制条件。
- 是否减少了强对比句式，尤其是“不是……而是……”和“是……而不是……”。
- 是否把抽象词换成了动作、对象、指标或时点。
- 语气是否自然，但没有变得油滑或过度口语化。
